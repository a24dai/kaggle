# Titanic: Machine Learning from Disaster

## csvのデータ
| *カラム名*  | *値*                        |
|-------------|-----------------------------|
| PassengerId | 乗客ユニークID              |
| Survived    | 生存フラグ                  |
| Pclass      | チケットクラス              |
| Name        | 乗客の名前                  |
| Sex         | 乗客の性別                  |
| Age         | 乗客の年齢                  |
| SibSp       | 同乗している兄弟/配偶者の数 |
| Parch       | 同乗している親/子供の数     |
| ticket      | チケット番号                |
| fare        | 料金                        |
| cabin       | 客室番号                    |
| Embarked    | 出港地                      |


### 補足

pclass  
- 1 : 上層クラス  
- 2 : 中層クラス  
- 3 : 下層クラス  

Embarked  
- C : Cherbourg  
- Q : Queenstown  
- S : Southampton
